**PortBaseTH** คือแพลตฟอร์มชุมชนสำหรับนักเรียนไทยที่กำลังเตรียม Portfolio เพื่อยื่นเข้ามหาวิทยาลัย รุ่นพี่ที่ผ่านกระบวนการนี้มาแล้วสามารถแชร์ผลงานจริง พร้อมประสบการณ์และแนวทางที่ทำให้ติด — เพื่อเป็นแรงบันดาลใจและข้อมูลให้รุ่นน้องที่กำลังก้าวตามมา

> *"เพราะทุก Portfolio มีเรื่องราว และทุกเรื่องราวมีคุณค่าที่ควรแบ่งปัน"*

```
┌─────────────────────────────────────────────────────────────┐
│  ◆  ARCHITECTURE HIGHLIGHTS                                  │
└─────────────────────────────────────────────────────────────┘
```

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Rendering** | Next.js Server Components | Zero-JS data fetching, SEO-ready |
| **Caching** | ISR (revalidate 60s) | Fast page loads via CDN edge |
| **Database** | PostgreSQL + RLS | Row-level security per user |
| **Storage** | Supabase Storage (Private) | Signed URLs for PDF access |
| **Auth** | Supabase Auth + OAuth | Google login + email/password |
| **Validation** | Zod + DOMPurify | Schema validation + XSS prevention |
| **Monitoring** | Sentry + Error Boundary | Real-time error tracking |
| **Rate Limit** | Upstash Redis | Sliding window, per-user limits |
| **SEO** | JSON-LD + Open Graph | Rich results + social previews |

```
┌─────────────────────────────────────────────────────────────┐
│  ◆  GETTING STARTED                                          │
└─────────────────────────────────────────────────────────────┘
```

**1 · Clone the repository**
```bash
git clone <repository-url>
cd portbaseth
```

**2 · Install dependencies**
```bash
npm install
```

**3 · Setup environment variables**

สร้างไฟล์ `.env.local` แล้วใส่ค่าต่อไปนี้
```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

# Upstash Redis (Rate Limiting)
UPSTASH_REDIS_REST_URL=
UPSTASH_REDIS_REST_TOKEN=

# Sentry (Error Tracking)
NEXT_PUBLIC_SENTRY_DSN=
SENTRY_DSN=
SENTRY_AUTH_TOKEN=
```

**4 · Run development server**
```bash
npm run dev
```

**5 · Open in browser**
```
http://localhost:3000
```
